## 2026-03-14 - Enhanced Contact Form Accessibility
**Learning:** Using `sr-only` labels provides essential context for screen readers in placeholder-only designs. Wrapping error messages in `role="alert"` and linking them with `aria-describedby` ensures immediate and clear feedback for assistive technologies.
**Action:** Always use `sr-only` labels for minimalist forms and ensure ARIA attributes (`aria-invalid`, `aria-required`, `aria-describedby`) are correctly wired with feedback components.
