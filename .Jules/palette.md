## 2025-03-20 - Accessible Forms with sr-only Labels
**Learning:** For accessible forms where only placeholders are desired, use `<Label>` components with the `sr-only` class to provide context to screen readers without changing the visual design. This is especially important for dark-themed, minimalist landing pages where a traditional label might feel cluttered.

**Action:** Always include a corresponding `<Label>` for every form input, and if the design doesn't call for a visible label, use the `sr-only` class to ensure the form remains usable for everyone.
